Tema 2

Instrumentación y Conexionado

Uso correcto del multímetro, lectura de señales alternas, osciloscopio, medición de velocidad con tacómetro y detección de puntos calientes con cámara termográfica.

Objetivos de aprendizaje

  • ✓ Conectar correctamente el multímetro para medir tensión, corriente y resistencia
  • ✓ Calcular Vp, Vpp, Vef, Vm y Vinst de una señal alterna
  • ✓ Leer amplitud, periodo y frecuencia en la pantalla de un osciloscopio
  • ✓ Calcular la constante de tiempo τ = R·C y el tiempo de carga de un condensador
  • ✓ Calcular la velocidad de un motor con el tacómetro y la fórmula de velocidad sincrónica
  • ✓ Interpretar imágenes termográficas para detectar puntos calientes en instalaciones
