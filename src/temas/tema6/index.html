Tema 6

Componentes Capacitivos

Condensadores: principio físico, capacitancia, carga y descarga RC, energía almacenada, asociaciones en serie y paralelo, clasificación y criterios de selección.

Objetivos de aprendizaje

  • ✓ Describir el principio físico de un condensador y calcular su capacitancia C = Q / ΔV
  • ✓ Calcular la capacitancia de un condensador de placas paralelas: C = ε · S / d
  • ✓ Analizar la carga y descarga RC: calcular τ = R·C y determinar V(t), I(t) en cualquier instante
  • ✓ Calcular la energía almacenada en un condensador: W = ½ · C · V²
  • ✓ Calcular la capacitancia equivalente en agrupaciones serie y paralelo de condensadores
  • ✓ Reconocer los principales tipos de condensadores y sus propiedades características
  • ✓ Seleccionar el tipo de condensador adecuado para una aplicación dada
