Práctica 07 · Media

Cuadro de vivienda

Montar un cuadro de mando y protección básico (IGA + ID + C1 + C2) en carril DIN según REBT ITC-BT-25.

⏱ Duración estimada: 90 min (2 sesiones) 👥 Individual o parejas 📚 Tema relacionado: Tema 4

Objetivos

  • ✓ Identificar los elementos de un cuadro de mando y protección: IGA, ID, PIAs
  • ✓ Dibujar el esquema unifilar del cuadro antes de montar
  • ✓ Montar correctamente el cuadro: IGA → ID → PIA C1 → PIA C2
  • ✓ Cablear la barra de neutros (N) y la barra de tierra (PE) correctamente separadas
  • ✓ Verificar el cuadro con multímetro antes de energizar

Material necesario

🏠 Caja de cuadro

Caja de superficie con carril DIN 35 mm y espacio para 12–18 módulos

⚡ IGA 2P 20A

Interruptor General Automático bipolar

🛡️ Diferencial 2P 25A 30mA

Interruptor diferencial clase AC tipo A

🔌 PIAs: 1×10A B (C1) + 1×16A B (C2)

Pequeños Interruptores Automáticos según ITC-BT-25

🔧 Barras N y PE + conductores

Peines de distribución, cable 1,5 mm² y 2,5 mm²

📏 Multímetro digital

Para verificación antes de energizar

Fundamento teórico — REBT ITC-BT-25

La instrucción técnica ITC-BT-25 del Reglamento Electrotécnico de Baja Tensión establece los circuitos mínimos para viviendas:

CircuitoUsoIn PIA (A)Sección (mm²)Puntos máx.
C1Alumbrado10 A (B)1,5 mm²30
C2Tomas uso general16 A (B)2,5 mm²20
C3Cocina y horno25 A (C)6 mm²2
C4Lavadora + lavavajillas + termo20 A (C)4 mm²3
C5Baño y cocina (tomas)16 A (B)2,5 mm²6
🚫
PELIGRO: El montaje del cuadro se realiza SIEMPRE con la alimentación DESCONECTADA. El IGA debe estar en posición OFF durante todo el montaje. Nunca trabajes con tensión en el carril.

Procedimiento

1

Dibujar el esquema unifilar

Antes de tocar ningún elemento, dibuja en tu cuaderno el esquema unifilar del cuadro: IGA (entrada) → ID (diferencial) → C1 (10A alumbrado) → C2 (16A tomas). Incluye barras N y PE separadas.

2

Fijar elementos en carril DIN

De izquierda a derecha: IGA (2P 20A) → ID 30mA → PIA C1 (B10A) → PIA C2 (B16A). Encaja todos en el carril. Instala la barra de neutros y la barra de tierra (separadas).

3

Cablear la barra de neutros (N, azul)

Conecta: N salida IGA → N entrada ID → N barra de neutros. Desde la barra de neutros, un hilo azul para cada circuito (C1 y C2).

4

Cablear la fase (L, marrón/negro)

L salida ID → entrada PIA C1 y entrada PIA C2. Salida PIA C1 = fase circuito alumbrado. Salida PIA C2 = fase circuito tomas.

5

Verificar con multímetro (sin tensión)

Con todo el cuadro en OFF: (a) Verifica continuidad L-entrada IGA → L-salida PIA C1/C2. (b) Verifica continuidad N-entrada → N-barra. (c) Verifica aislamiento: entre L y N de salida debe ser OL. (d) Solo si todo correcto, el profesor autoriza energizar.
